# 10. Example Comparison
* JavaScript Code:
```javascript
function add(a, b) {
    return a + b;
}
console.log(add(5, "10")); // Outputs "510" (string concatenation)
```
* TypeScript Code:
```typescript
function add(a: number, b: number): number {
    return a + b;
}
// console.log(add(5, "10")); // Error: Argument of type 'string' is not assignable to parameter of type 'number'.
```

# Conclusion
* **Choose JavaScript** if you need speed, flexibility, and simplicity for small to medium-sized projects.
* **Choose TypeScript** for larger, scalable applications where type safety, maintainability, and collaboration are essential.
