# Definition
* Component is a reusable, self-contained building block of a user interface.
* Think of components as functions or classes that return UI elements (usually written in JSX, a syntax similar to HTML)

# Key Concepts of React Components:
# 1. Types of Components
1. **Function Components :**  The modern standard. These are just functions that return JSX
```jsx
import React from 'react';

function Header() {
  return <h1>Welcome!</h1>;
}

export default Header; // üîÅ Exporting the component

==================

function Welcome(props) {
  return <h1>Hello, {props.name}</h1>;
}
```
2. **Class Components :** (older style)
```jsx
class Welcome extends React.Component {
  render() {
    return <h1>Hello, {this.props.name}</h1>;
  }
}
```
# 2. JSX (JavaScript XML)
* JSX lets you write HTML-like code inside JavaScript
  ```jsx
     return <div>Hello World</div>;
  ```

# 3. Props (Properties)
* Inputs to components. Passed like attributes in HTML
  ```jsx
  <Welcome name="John" />
  ```
# 4. Stage
* Internal data, a component can manage and change
  ```jsx
  const [count, setCount] = useState(0);
  ```
