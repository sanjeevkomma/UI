| #Version  | #Release Date |#Features |
| :--- | :--- | :--- |
| 0.3.0 | July 2013 |Initial public release; introduced the concept of components and a virtual DOM |
| 0.14.0 | October 2015 |Split the React library into two packages: react for components and react-dom for DOM rendering. |
| 15.0.0 | April 2016 |Improved performance; introduced stateless functional components. |
| 16.0.0 | September 2017 |Introduced Fiber architecture for incremental rendering; added error boundaries. |
| 16.8.0 | February 2019 |Introduced Hooks API, allowing state and side effects in functional components. |
| 17.0.0 | October 2020 |Focused on making upgrades easier; no new developer-facing features. |
| 18.0.0 | March 2022 |Introduced concurrent features like automatic batching, transitions, and startTransition |
| 18.3.0 | April 2024 |Added warnings for deprecated APIs to prepare for React 19. |
| 19.0.0 | December 2024 |Introduced React Actions, allowing startTransition to accept async functions; various performance improvements |
