# Why do we use Next.js?
* Problems with plain React (SPA)
  * Slow first page load
  * Poor SEO (content rendered only in browser)
  * Manual setup for routing, performance, and API calls

# What Next.js solves
* Faster load times
* Better SEO
* File-based routing
* Backend APIs inside frontend app
* Easy deployment
