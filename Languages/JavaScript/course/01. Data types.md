# Example
```javascript
// Primitive Types
let name = "Alice";   // String
let age = 25;         // Number
let isHappy = true;   // Boolean
let emptyValue = null; // Null
let notDefined;       // Undefined
let bigNumber = 123456789012345678901234567890n; // BigInt
let uniqueId = Symbol('id'); // Symbol

// Reference Types
let person = { name: "Alice", age: 25 }; // Object
let numbers = [1, 2, 3, 4, 5];           // Array
function greet() { console.log("Hi"); }  // Function
let today = new Date();                  // Date
let pattern = /abc/;                     // RegExp
```
# const in JavaScript
* const is NOT a data type.
* const is a keyword for declaring variables.
* It means: You cannot reassign the variable to a new value

# Example for const in JavaScript
```javascript
const name = "Alice";  // constant string
const age = 30;        // constant number
const isHappy = true;  // constant boolean

name = "Bob";  // ❌ ERROR! You can't change a const variable.

const person = { name: "Alice" };
person.name = "Bob";   // ✅ Allowed: modify property
person = {};           // ❌ ERROR: can't reassign

const numbers = [1, 2, 3];
numbers.push(4);       // ✅ Allowed: modify array
numbers = [5, 6, 7];   // ❌ ERROR: can't reassign

```
