# Definition
* ECMAScript = European Computer Manufacturers Association Script
* It is a standardized scripting language specification that serves as the foundation for several programming languages, the most well-known being JavaScript

# Versions
# 1. ECMAScript 1 (1997)
* First edition (baseline version)
# 2. ECMAScript 2 (1998)
* Editorial changes, no new features
# 3. ECMAScript 3 (1999)
* Added regular expressions
* Try/catch exception handling
* Better string handling
* New control statements (do...while, switch)
* delete, typeof, and in operators
# 4. ECMAScript 4 (Abandoned)
* Proposed major changes but never finalized
* Led to split between ECMAScript and ActionScript (Flash)
# 5. ECMAScript 5 (2009)
* Strict mode ("use strict")
* Array.isArray()
* Object.create(), Object.defineProperty()
* forEach, map, filter, reduce
* JSON support
* Immutable object methods: Object.freeze(), etc.
# 5.1. ECMAScript 5.1 (2011)
* Aligns with ISO/IEC standard
* Minor fixes and editorial clarifications
# 6. ECMAScript 6 / ES2015 (2015)
* let, const (block-scoped variables)
* Arrow functions () => {}
* Classes, Modules (import, export)
* Template literals `Hello ${name}`
* Default parameters
* Destructuring
* Promises
* Map, Set, WeakMap, WeakSet
* Rest (...args) and Spread operators
# 7. ECMAScript 2016 (ES7)
* Array.prototype.includes()
* Exponentiation operator (**)
# 8. ECMAScript 2017 (ES8)
* async/await
* Object.entries(), Object.values()
* String padding: padStart(), padEnd()
* SharedArrayBuffer, Atomics
# 9. ECMAScript 2018 (ES9)
* Rest/spread for objects
* Promise.prototype.finally()
* Asynchronous iteration (for await...of)
* RegExp improvements (dotAll, named capture groups)
# 10. ECMAScript 2019 (ES10)
* Array.prototype.flat(), flatMap()
* Object.fromEntries()
* Optional catch binding (try { ... } catch { ... })
* String.prototype.trimStart(), trimEnd()
# 11. ECMAScript 2020 (ES11)
* Optional chaining (obj?.prop)
* Nullish coalescing (??)
* Promise.allSettled()
* globalThis
* Dynamic import()
* BigInt (123n)
* String.prototype.matchAll()
# 12. ECMAScript 2021 (ES12)
* Logical assignment operators (&&=, ||=, ??=)
* Numeric separators (1_000_000)
* String.prototype.replaceAll()
* Promise.any()
* WeakRefs and FinalizationRegistry
# 13. ECMAScript 2022 (ES13)
* Top-level await in modules
* Array, TypedArray methods: at()
* Class fields and private methods (#myField)
* Object.hasOwn()
# 14. ECMAScript 2023 (ES14)
* Array grouping methods: groupBy(), groupByToMap()
* Symbol.prototype.description
* Array.prototype.toSorted(), toReversed(), with()
* RegExp v flag (for set notation and property escapes)
# 15. ECMAScript 2024 (ES15)
* Object.groupBy() and Map.groupBy()
* Promise.withResolvers()
* Temporal API Enhancements
* Regular Expression /v Flag
* Pipeline Operator (|>)
* Pattern Matching (Proposed)
* Async Context Tracking
# 16. ECMAScript 2025 (ES16)
* Pattern Matching
* Immutable Data Structures: Records and Tuples
* Pipeline Operator (|>)
* Enhanced Asynchronous Iteration
* New Set Methods
* Iterator Helpers
* JSON Modules
* Regular Expression Enhancements
* Promise.try()
* Float16Array and Related Methods
